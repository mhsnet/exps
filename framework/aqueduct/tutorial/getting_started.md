## <span id="installation">安装</span> [Top]
```
$ pub global activate aqueduct --- 安装Aqueduct命令行工具
```

## <span id="creating-project">创建项目</span> [Top]
```
$ aqueduct create dart_study_server --- 创建Aqueduct项目
$ aqueduct serve --- 启动项目
```
