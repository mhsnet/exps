## <span id="basics">基础</span>
> 1. 微服务使用与HTTP不同传输层

### <span id="basics-install">安装</span>
```
$ npm add @nestjs/microservices
```

### <span id="basics-overview">概述</span>
> 1. 支持多个内置传输器，它们基于请求响应和事件
> 2. 请求响应不支持持续流
> 3. 可与事件(单向通信)一起使用
