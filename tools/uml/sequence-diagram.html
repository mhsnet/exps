信息

  1. 实例间消息传递
  2. header/footer/title
  3. group --- 信息自定义

代码：

@startuml
participant A
participant B
A -> B: req1
group if
  group if(Login successfully)
  A <- B: rt1
  end
  group else if(Login failed)
  A <- B: rt2
  end
  group else
  A <- B: rt3
  end
end
@enduml

分组

用(box|end box)添加分组|标题|背景色

代码：

@startuml
participant A #red
box "B And C" #silver
	participant B #green
  participant C #blue
end box
hide footbox
@enduml
